object-fit : fill | contain | cover | none | scale-down;

이미지나 비디오 등의 가로, 세로 비율을 유지하면서 크기를 조절할 수 있다

object-fit : fill

콘텐츠의 원래 비율을 무시하고 요소 전체 영역을 채움. 기본값

object-fit : contain

비율을 유지하면서 요소의 전체 영역에 맞출 수 있을만큼 확대. 영역을 모두 채우지 못할 수도 있다.

object-fit : cover

비율을 유지하면서 요소의 전체 영역을 채울 수 있을만큼 확대. 영역을 채우기 위해 콘텐츠의 일부가 잘릴 수 있다.

object-fit : none

콘텐츠의 원래 크기 유지

object-fit : scale-down

none과 contain 중 콘텐츠 크기가 더 작아지는 값 선택
